# DRF CRUD APIs (FBV and  CBV)
FBV -> function based views
CBV -> class based views

This repo demonstrates CRUD APIs using Django REST Framework.

## Apps:
- Student API → Function-Based Views (FBV)
- Employee API → Class-Based Views (CBV) (soon)

## Purpose:
To understand differences between FBV and CBV in DRF.

## Setup:
pip install -r requirements.txt (IMP: to install dpendencies , don't forget to activate virtual env :> )
python manage.py runserver
